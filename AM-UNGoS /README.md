# AM-UNGoS

## Installation Guide

- Install Go 1.18 (`curl -L https://git.io/vQhTU | bash`)
- Install sqlite3 (`apt install sqlite3`)
- Git clone this repository
- Build and run the executable or use `go run main.go`

## Webpage Cache Issues
If you encounter strange issues related to being logged in/out, open Developer Tools in your browser, go to Network and turn on Disable Cache. While Developer Tools is open, you can force your cache to update properly when you navigate there. Idk how to fix this.
